<!-- ABOUT THE PROJECT -->
## About The Project

<!-- [![Product Name Screen Shot][product-screenshot]](https://example.com) -->

Simple Chip8 Emulator building with .NET core 8 and Avalonia UI inspired by [Queso Fuego](https://www.youtube.com/@QuesoFuego)'s Chip8 tutorial on [Youtube](https://www.youtube.com/watch?v=YvZ3LGaNiS0&list=PLT7NbkyNWaqbyBMzdySdqjnfUFxt8rnU_).

<p align="right">(<a href="#readme-top">back to top</a>)</p>
